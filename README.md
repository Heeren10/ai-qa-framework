# AI QA Framework 🤖

An intelligent AI-powered QA system that:
- Understands business logic
- Generates test scenarios
- Simulates human QA testing

## 🚀 Features
- Logic Alignment Agent (Phase 1)
- Business understanding from URL + BRD
- Personas, flows, and logic extraction

## 🛠 Tech Stack
- FastAPI
- Gemini API
- Playwright (coming next)
- JavaScript UI
